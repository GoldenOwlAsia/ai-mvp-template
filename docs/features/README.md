# Features

Per-feature specs for the **active** product. Empty in the base template.

| Path | When |
|---|---|
| `docs/_blank/feature-template.md` | Template for a new feature file |
| `docs/_example-taskflow/features/` | Filled Kanban example (tutorial / REF) |
| `docs/features/<name>.md` | **Your app** — created via `prompts/init-product.md` or copied from the example |

Do not commit domain feature specs to the base unless they belong to your product.
